- Játékos keresése => minden karakter megengedet
- Alkalom neve => minden, lehet üres
- Körök száma => csak számok, nem lehet üres
- Csapatok száma => csak számok, nem lehet üres

	- ha 0-ra állítja, akkor kipipálja magát
